Le dossier Img contient tous les écrans du jeu qui sont les fichiers IMG du Daggerfall classique, transformés au format PNG pour être affichées par DFU.

Le dossier CifRci contient des images .RCI qui sont les boutons gris (OUI, NON...) qui se trouvent dans certaines boites de dialogue de Daggerfall. Là aussi les images ont été passées au format PNG.
