Le fichier Internal_Strings.csv contient de nombreux textes du jeu.
Le fichier Internal_Locations.csv est un export de MAPS.BSA qui contient les noms des 15200 lieux présents sur la carte du jeu. Tous les textes de ce fichier ont été francisés mais ce n'est pas encore la version définitive qui est en cours de production avec Pango. Si on veut rester sur les noms anglais originaux, alors il ne faut pas installer ce fichier dans StreamingAssets/Text.
Le fichier Internal_Quests contient l'intégralité des textes de toutes les quêtes du jeu. CE fichier peut encore être modifié par Interkarma à l'avenir, mais pour l'instant il fonctionne avec la 0.15.1 bêta.
Les fichiers .txt traduisent l'interface d’accueil de DFU.
Il en existe un quatrième, DialogShortcuts.txt qui définit les raccourcis clavier. Ce fichier devrait peut-être être modifié pour adapter certaines touches au français -> à voir à l'usage.
