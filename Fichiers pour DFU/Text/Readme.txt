Le fichier Internal_Strings.csv contient de nombreux textes du jeu.
Le fichier Internal_Locations.csv est un export de MAPS.BSA qui contient les noms des 15200 lieux présents sur la carte du jeu. Ce fichier contient les noms françisés. Si on veut rester sur les noms anglais originaux, alors il ne faut pas installer ce fichier dans StreamingAssets/Text.
Les fichiers .txt traduisent l'interface d’accueil de DFU.
Il en existe un quatrième, DialogShortcuts.txt qui définit les raccourcis clavier. Ce fichier devrait peut-être être modifié pour adapter certaines touches au français -> à voir à l'usage.
