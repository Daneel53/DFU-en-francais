Le fichier Internal_Strings.csv contient des textes du jeu et des listes qui étaient pour la plupart enfermés dans le fichier FALL.EXE de Daggerfall.
Le fichier Internal_RSC.csv contient la grande majorité des textes du jeu, sauf ceux contenus dans les quêtes.
Le fichier Internal_Quests.csv contient l'intégralité des textes de toutes les quêtes du jeu. Ce fichier de 23000 lignes sera peut être découpé et modifié par Interkarma dans une version de DFU ultérieure à la 0.15.1.
Le fichier Internal_Locations.csv est un export de MAPS.BSA qui contient les noms des 15200 lieux présents sur la carte du jeu. Tous les textes de ce fichier ont été francisés avec une procédure écrite par Pango. Si on veut rester sur les noms anglais originaux, alors il ne faut pas installer ce fichier dans StreamingAssets/Text.
Le fichier Internal_Flats.csv contient la liste de tous les types de PNJ qu'il est possible de rencontrer dans le jeu.
Les trois fichiers .txt MainMenu, GameSettings et ModSystem traduisent l'interface d’accueil de DFU.
Il en existe un quatrième, DialogShortcuts.txt qui définit les raccourcis clavier. Ce fichier devrait peut-être être modifié pour adapter certaines touches au français -> à voir à l'usage.
